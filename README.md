# Portfólio Allan Delon Pedro

Este é um site de portfólio profissional desenvolvido para apresentar minhas habilidades, projetos e experiência como desenvolvedor backend e full stack.

## Tecnologias Utilizadas

O site foi construído utilizando exclusivamente **HTML, CSS e JavaScript puro**, sem frameworks ou bibliotecas externas, garantindo leveza, performance e compatibilidade.

- **HTML**: Estrutura semântica das páginas
- **CSS**: Estilos modernos com tema escuro, animações suaves e design responsivo
- **JavaScript**: Funcionalidades interativas como navegação suave e animações de entrada

## Como Visualizar

### Opção 1: Localmente
1. Clone o repositório:
   ```
   git clone https://github.com/AllDPedro/curriculum-site.git
   ```
2. Abra o arquivo `index.html` em um navegador web.

### Opção 2: GitHub Pages
O site está hospedado no GitHub Pages e pode ser acessado em: [https://alldpedro.github.io/curriculum-site/](https://alldpedro.github.io/curriculum-site/)
